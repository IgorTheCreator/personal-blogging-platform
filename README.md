## Как запустить приложение

- Установите зависимости

```
npm install
```

- Создайте файл .env и добавьте параметры подключения к базе данных. Пример можно найти в .env.sample

- Выполните миграции

```
npm run migrate
```

- Запуск приложения

```
npm run dev
```
